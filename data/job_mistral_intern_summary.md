## 1) Application requirements (from the posting, paraphrased)

- To apply, candidates must provide a GitHub repository and complete a project.
- Suggested project directions include:
  - a Next.js chat app using Mistral’s public API,
  - a Python project (e.g., using vLLM or similar),
  - or a flexible topic that uses Python/FastAPI or Next.js/TypeScript and incorporates Mistral’s SDK.
- They will pay particular attention to:
  - a detailed README,
  - and making the project easy to test.

---

## 2) Role summary (paraphrased)

- You’d work with the engineering team and learn specifics of AI software.
- You’d build both internal and external tooling.
- Location: Paris (on-site).
- Internship duration: 4–6 months; they prioritize candidates able to join full-time after a successful internship.

---

## 3) What you will do (paraphrased)

- Build user-facing internal/external apps powered by Mistral models (examples: chatbot, search, document answering).
- Instrument products with developer-facing tools (examples: dashboards, evaluation interfaces).

---

## 4) About you (paraphrased)

- Strong full-stack skills (Python / TypeScript / JavaScript) **or** infrastructure skills (Kubernetes / CI).
- Ability to create smooth, high-quality user experiences for both developers and end users.
- Appetite for building AI solutions with chat APIs, embedding APIs, etc.
- Ability to adapt quickly and evolve in a fast-changing environment.

---

## 5) Interview process (paraphrased)

- Submit GitHub + project
- Coding interview
- System design interview
